A safety layer for every false positive

🛰️

State machine

IDLE -> ACTIVE -> CONFIRMING -> COOLDOWN. Actions pass through explicit confirmation.

🎯

Multi-frame stability

A gesture only counts after repeated matching frames and enough confidence.

🤝

Multimodal confirmation

Gaze plus gesture. Critical actions do not rely on one signal alone.

✋

Always-on pause

A closed fist can pause the system from any state.

👁️

Personal calibration

Ridge regression trained with an aim-trainer style minigame.

🔒

Local by design

Offline speech support and local camera processing. Privacy is the default.

Three signals, one control loop

01

Gaze

FaceMesh + iris + nose/head cues -> personal ridge regression -> screen (x, y).

02

Gestures

MediaPipe Hands -> index click · middle right click · two fingers double click · fist pause.

03

Voice

Local Whisper commands for activation, pause, click, zoom and navigation.
