Una capa por cada falso positivo

🛰️

Máquina de estados

IDLE → ACTIVE → CONFIRMING → COOLDOWN. Nada se ejecuta sin pasar por confirmación.

🎯

Estabilidad multi-frame

Un gesto sólo cuenta tras N frames consecutivos coincidentes y confianza media.

🤝

Confirmación multimodal

Mirada + gesto. Nunca una sola modalidad para acciones críticas.

✋

Pausa siempre disponible

Puño cerrado 1 s pausa el sistema en cualquier estado.

👁️

Calibración personal

Ridge regression entrenada con un minijuego tipo aim trainer.

🔒

100 % local

Whisper offline, ningún frame sale del equipo. Privacidad por diseño.

Tres modalidades, un solo flujo

01

Mirada

FaceMesh + iris → ridge regression personalizada → (x, y) en pantalla.

02

Gestos

MediaPipe Hands → 👍 click · 👎 escape · pinch zoom · puño pausa.

03

Voz

Whisper local para dictado y comandos largos (Phase 3).
